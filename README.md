## 📌 Project Overview

This repository contains a public-safe version of my portfolio website. It presents my technical background, selected projects, certifications, and professional experience in a clear and responsive interface.

## ✨ Main Features

- Responsive design for desktop and mobile
- Cybersecurity and networking project showcase
- Software and web development work
- Certifications gallery
- Professional experience and skills sections
- Direct link to the live production website
- Privacy-conscious public source package

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Responsive Web Design
- Git and GitHub

## 🔐 Security and Privacy

This public repository intentionally excludes:

- Passwords, API keys, tokens, and private keys
- `.env` files and production configuration
- Student IDs and government identification
- Private client documents and database backups
- Unreviewed archives and lab credentials
- Server-side contact configuration
